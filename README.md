Alhamdulillah ‚Äî I developed a **Food Delivery App** using **Flutter** (frontend), **Django REST** (backend / orders), **Firebase** (push notifications), and **Stripe** (payments).

## üöÄ Tech Stack

- **Frontend:** Flutter

---

## ‚ú® Features

- User authentication using access token & profile management  
- Browse food items & menu details  
- Add to cart & place orders

- Real-time push notifications (FCM)  
- Secure Stripe payment checkout  
- Location tracking for delivery
